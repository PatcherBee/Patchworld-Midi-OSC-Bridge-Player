# Patchworld-Midi-OSC-Bridge-Player
build using JUCE Framework. This app runs a local OSC server over ipv4 and bridges pc to patchworld (vice versa). Great for DAW passthrough to Patchworld (vice versa) Allow install from unknown source and allow network access (local ip only). More to come when ready. Uses Ableton Link repository (find projects for JUCE/Link on github)
